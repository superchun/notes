## 光标移动

`cmd` + `←`	  移动至行首

`cmd` + `→`	  移动至行尾

`fn` + `↓` 	   向下翻页

`fn` + `↑` 	   向上翻页

`option` + `←`    向前跳一个词

`option` + `→` 	向后跳一个词

`ctl` + `p`     光标向上移动一行

`ctl` + `n`     光标向下移动一行

`ctl` + `b` 	光标向前移动一个字母

`ctl` + `f`  	光标向后移动一个字母

`ctl` + `e`      光标移动到行尾

`ctl` + `a`      光标移动到行首

## 选取

`option` + `shift` + `←` 	向前选中一个单词

`option` + `shift` + `→`     向后选中一个单词

`cmd` + `shift` + `→`	    从当前光标选中到行尾

`cmd` + `shift` + `←` 		从当前光标选中到行首

## 删除

`fn` + `delete`      向后删除

`option` + `delete`     向前删除一个单词

`fn` + `option` + `delete`     向后删除一个单词

`cmd` + `delete`	 从当前光标删除到行首

`fn` + `cmd` + `delete`     从当前光标删除到行尾
